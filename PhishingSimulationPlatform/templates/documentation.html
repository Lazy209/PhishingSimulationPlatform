1. Overview

PhishGuard is a phishing simulation platform for cybersecurity awareness training, built as part of the IBM SkillBuild Internship programme. It uses Python Flask for the web server, smtplib for email delivery, and SQLite for engagement metrics.

Tech Stack: Python 3  ·  Flask  ·  SQLite  ·  smtplib  ·  Jinja2  ·  HTML / CSS

Key Constraint: Passwords entered on the fake login page are never stored — the field is read once and immediately discarded.

4. User Workflow

  1. Admin runs send_campaign.py or user clicks Try Demo
  2. Recipient receives email or opens demo tracking link
  3. Click logged → fake Microsoft login page shown
  4. User submits form → password discarded, event logged
  5. Micro-learning page explains all missed phishing indicators
  6. Admin views metrics at /admin

5. Security & Ethics

  • Passwords are read and immediately discarded — never logged or persisted
  • Recipient emails stored as SHA-256 hash prefix — not plaintext
  • Unique cryptographic token (secrets.token_urlsafe) per recipient
  • Training disclaimers visible on all simulation pages
  • Authorized lab use only — not for real unauthorized phishing attacks

6. How to Run

cd PhishingSimulationPlatform
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py

Open http://127.0.0.1:5000 or double-click run.bat.

Admin login: username admin  /  password set in .env
